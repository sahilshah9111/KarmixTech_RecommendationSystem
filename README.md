# 🎬 Hybrid Movie Recommendation System

A personalized movie recommendation engine built using Content-Based Filtering, Collaborative Filtering (SVD), and a Hybrid Recommendation approach.

## 📌 Overview

This project recommends movies based on:

- User behavior and ratings
- Movie genres and tags
- Hybrid recommendation strategy

The system combines:

- Content-Based Filtering (TF-IDF + Cosine Similarity)
- Collaborative Filtering (SVD)
- Hybrid Recommendation Model
- Cold Start Handling

---

## 📂 Dataset

MovieLens Latest Small Dataset

Files Used:

- movies.csv
- ratings.csv
- tags.csv
